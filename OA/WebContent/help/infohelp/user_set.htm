浏览设置：

 

此模块的主要功能是设置信息发布单位在信息浏览树中的排序方式，及信息浏览的用户常用栏目。

 

信息发布单位排序：

点击左侧页面树状结构中“组织分类”名称在右侧信息发布单位排序页面中列出此组织分类下的信息发布单位，及其排序方式，选中列表框中要排序的信息发布单位，点击“上移”、“下移”箭头，改变其排序顺序，之后点击“保存”按钮保存排序方式。

 

信息浏览常用栏目设置：

在“信息浏览默认栏目设置页面”的首页栏目列表框中会列出用户的信息浏览首页常用栏目（如果用户没有设置常用栏目，那么在信息浏览首页将显示系统默认的栏目）。

点击左侧页面树状结构中“信息发布单位”名称在右侧“信息浏览默认栏目设置页面”的“可选择栏目列表框”中列出此信息发布单位下的信息栏目，选中要添加的栏目，点击“移入”箭头添加常用栏目。

在“信息浏览默认栏目设置页面”的“首页栏目列表框”中选中要去除的常用栏目，点击“移出”箭头去掉此常用栏目。

可以通过“上移”、“下移”箭头对常用栏目进行排序。

设置好常用栏目后点击“保存”按钮保存设置，点击“恢复默认”按钮可以将常用栏目设置恢复为系统默认的栏目。
