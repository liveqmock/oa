网络文件夹

 

点击文件传递－>网络文件夹，进入网络文件夹的模块的界面如下图所示。

 

界面左部是网络文件夹的树状结构：分为个人文件夹：个人文件夹是用户创建的文件夹；共享文件夹：

共享文件夹是其他用户给你共享使用的文件夹。通过点击树状结构可以在右侧的界面中显示对应的文件夹信息列表。

从右侧的界面中多个按钮中可以看到网络文件夹的基本操作（新建、上传、下载、重命名、删除、共享、消除共享、移动、保存）

 

 

 

 

网络文件夹 [ 新建 ] 功能：

功能简述：在一个文件夹下建立新的文件夹。

操作步骤：在左面的结构树中点击相应的文件夹或者在右侧的界面中点击文件夹名称，进入相应的文件夹中，点击

主界面中的“新建”按钮，进入下图所示新建操作的界面。在文件夹名称中填入新建的文件夹的名称，如果没有填写

系统将提示填写，在文件夹描述中填入该文件夹的描述，该处为可选项。按要求填完后点击“添加”按钮就可以

完成新建操作；点击“返回”按钮，放弃文件夹的添加操作并返回前一界面。

 

 

 

网络文件夹 [ 上传 ] 功能：

功能简述：将一个文件上传到相应的文件夹下。

操作步骤：在左面的结构树中点击相应的文件夹或者在右侧的界面中点击文件夹名称，进入相应的文件夹中，点击

主界面中的“上传”按钮，进入下图所示上传文件操作的界面。点击“游览”按钮在弹出的对话框中选择要

上传的文件，如果没有选择文件系统将提示填写，在文件描述中填入该文件夹的描述，该处为可选项。

按要求填完后点击“上传”按钮就可以完成上传操作；点击“返回”按钮，放弃文件的上传操作并返回前一界面。

 

 

网络文件夹 [ 下载 ] 功能：

功能简述：将一个文件在客户端打开，或者保存到客户机上。

操作步骤：在左面的结构树中点击相应的文件夹或者在右侧的界面中点击文件夹名称，进入相应的文件夹中，

在右侧的界面中单击某个文件的名称，出现如果所示的弹出文件下载对话框，单击打开按钮系统根据文件名称自动打开该文件，

如果点击保存按钮就将文件保存到相应的目标地址中。下图所示即为文件下载操作的界面。

 

 

网络文件夹 [ 重命名 ] 功能：

功能简述：改变一个文件或文件夹的名称。

操作步骤：在左面的结构树中点击相应的文件夹或者在右侧的界面中点击文件夹名称，进入相应的文件夹中，选择一个

要改名的文件夹。如果多选或者不选系统会提示。点击主界面中的“重命名”按钮，进入下图所示重命名操作的界面。

在文件夹名称填写要改成的文件夹名称，如果此项为空系统会提示。在文件描述中填入该文件夹的描述，该处为可选项。

按要求填完后点击“修改”按钮就可以完成上传操作；点击“返回”按钮，放弃文件的上传操作并返回前一界面。

 

 

 

 

网络文件夹 [ 删除 ] 功能：

在左面的结构树中点击相应的文件夹或者在右侧的界面中点击文件夹名称，进入相应的文件夹中，选择一个或多个

要删除的文件夹。如果不选系统会提示。点击主界面中的“删除”按钮，下图所示重命名操作的界面。

 

 

 

网络文件夹 [ 共享 ] 功能：

功能简述：将一个文件夹共享给其他人。

操作步骤：在左面的结构树中点击相应的文件夹或者在右侧的界面中点击文件夹名称，进入相应的文件夹中，选择一个

要共享的文件夹。如果多选或者不选，系统会提示。点击主界面中的“共享”按钮，进入下图所示共享操作的界面。

如下图所示：

 

点击“从地址簿选择”按钮可以进入地址簿选择界面（关于地址簿的操作，请参阅地址簿操作部分）

 

在以共享的人员列表中可以选择一个或多个人员进行删除操作，在共享名填写文件夹的共享名称，缺省为文件夹的原名，

按要求填完后点击“确定”按钮就可以完成共享操作；点击“返回”按钮，放弃文件的上传操作并返回前一界面。如果

完成共享操作，文件夹标识上就会出现共享标识。

 

 

网络文件夹 [ 取消共享 ] 功能：

功能简述：将一个已经共享的文件夹取消共享。

操作步骤：在左面的结构树中点击相应的文件夹或者在右侧的界面中点击文件夹名称，进入相应的文件夹中，选择一个或多个

要取消共享的文件夹。如果不选系统会提示。点击主界面中的“取消共享”按钮，就可完成取消共享操作，文件夹标识

上共享标识也会消失，下图所示重命名操作的界面

 

 

网络文件夹 [ 移动 ] 功能：

    功能简述：将一个文件或文件夹移动到其他文件夹下。

操作步骤：在左面的结构树中点击相应的文件夹或者在右侧的界面中点击文件夹名称，进入相应的文件夹中，选择一个

要移动的文件夹。如果多选或者不选，系统会提示。点击主界面中的“移动”按钮，进入下图所示移动操作的界面。

在弹出的对话框中点选移动的目标文件夹，选择的文件夹的名称会在显示出来，如果没有选择，系统将提示填写；如果选择

的目标文件夹和原文件夹相同，系统也会提示在文件夹描述中填入该文件夹的描述，该处为可选项。按要求选择后点击“确定”

按钮就可以完成文件夹的移动操作；点击“取消”按钮，放弃文件夹的添加操作并返回前一界面。

 

 

网络文件夹 [ 保存 ] 功能：

    功能简述：将一个文件或文件夹拷贝到其他文件夹下。

操作步骤：在左面的结构树中点击相应的文件夹或者在右侧的界面中点击文件夹名称，进入相应的文件夹中，选择一个

要移动的文件夹。如果多选或者不选，系统会提示。点击主界面中的“保存”按钮，进入下图所示移动操作的界面。

在弹出的对话框中点选移动的目标文件夹，选择的文件夹的名称会在显示出来，如果没有选择，系统将提示填写；如果选择

的目标文件夹和原文件夹相同，系统也会提示在文件夹描述中填入该文件夹的描述，该处为可选项。按要求选择后点击“确定”

按钮就可以完成文件夹的拷贝操作；点击“取消”按钮，放弃文件夹的添加操作并返回前一界面。
